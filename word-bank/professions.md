I actually missed the first class in which this was discussed. Here is a list of professions we've encountered along the way and with an example, context-dependent sentence that amounts to saying someone (for which the context is clear if not defined here) is that profession. This will be useful for drilling down when to use 이에요 or 예요. In the case of this basic "is" verb, it conjugates to the noun.

| **직업 (한글)**| **Profession** | **Basic Context-Dependent Example Sentence** |
|:------:|:------------:|:-------------------:|
| 회사원 | Employee (office worker) | 회사원이에요 | 
| 대학생 | Student | 대학생이에요 |
| 의사 | Doctor | 의사예요 |
| 경찰 | Police (cop) | 경찰이에요 |
| 선생님 | Teacher | 선생님이에요 |
| 가수 | Singer | 가수예요 |
| 요리사 | Chef | 요리사예요 |
| 교수 | Professor | 교수예요; 스티브 씨는 교수예요|
